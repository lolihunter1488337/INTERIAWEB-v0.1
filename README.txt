INTERIA! RECORDS — сайт на React + Tailwind + Framer Motion
============================================================

ЛОГОТИП (1 действие):
  Положи свой хром-лого сюда:  public/logo.png
  На живом сервере он появится сразу (hot reload).
  Для собранной версии — положи также в dist/logo.png

ЗАПУСК ЖИВОГО РЕЖИМА (с анимациями, hot reload):
  1) открой терминал в папке interia-app
  2) npm run dev
  3) открой http://localhost:5173/

СОБРАТЬ ФИНАЛЬНУЮ ВЕРСИЮ (для хостинга):
  npm run build   ->  готовый сайт лежит в папке dist/

СТЕК / ЧТО ИСПОЛЬЗОВАНО:
  - React + Vite
  - Tailwind CSS (как в 21st.dev)
  - Framer Motion: variants, stagger, whileInView, spring,
    useScroll/useTransform (параллакс), AnimatePresence,
    motion values (магнитные кнопки, тилт-карточки, счётчики)

ДАЛЕЕ:
  - реальная отправка формы (Formspree)
  - заливка на GitHub Pages / Cloudflare + свой домен
